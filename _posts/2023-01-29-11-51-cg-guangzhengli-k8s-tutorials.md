### 如何安装使用

安装方法：

1. 安装 Git，如果您的系统中没有安装 Git，请先安装。
2. 使用 Git clone 命令下载项目：```git clone https://github.com/guangzhengli/k8s-tutorials.git```
3. 进入项目目录并按照给定的说明安装和配置所需的依赖。

具体的 demo 使用代码可以在项目中查看，每个示例都有详细的注释和说明。
