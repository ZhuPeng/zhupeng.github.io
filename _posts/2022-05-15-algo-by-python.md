大家好。

常见的算法有非常的多，如果都要自己去实现一遍的话难度非常的大，对于我们正常工作的同学来说，很多算法都只是需要了解即可，看一看其他人的实现就行。

今天要推荐的是一个涵盖非常全的算法开源项目，该项目使用 Python 实现，实现了超过 100+ 以上的常见算法，选择 Python 的一个好处是比较容易写，同时即使你没有 Python 的基础，也是能够看懂算法的逻辑的，两全其美。
