做前端开发的同学应该都知道 Express 框架吧，Express 是一个保持最小规模的灵活的 Node.js Web 应用程序开发框架，为 Web 和移动应用程序提供一组强大的功能。

今天要推荐的项目就是受 Express 项目的启发用 Go 开发的 Web 框架：fiber，它简洁、性能强大，遵从极简主义的思想和 UNIX 的设计原则，对于 Go 语言的初学者也能很快的学习和使用。
