今天给大家推荐一个非常好用的表格前端组件库，Grid.js：支持所有主流的浏览器和前端框架，有如下特点：

- 实现上简单且轻量化，库只有 12kb 大小
- 依赖简单，Grid.js 可以在任何的 JavaScript 框架中使用（如 React, Angular, Preact, VanillaJS)
- 使用 TypeScript 开发
- 支持所有主流的浏览器和 IE11+，使用 MIT 开源许可
