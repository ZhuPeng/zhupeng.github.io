以下是项目的介绍：

主要技术实现：vue+element+axios+vuex

按如下方式可以完成项目的启动运行：

3.1、拉取项目 git clone -b dev https://github.com/www-chao-fun/chaofun-front.git

3.2、进入目录 cd chaofun-front

3.3、安装依赖 npm install 或者cnpm install	

3.4、运行项目 npm run dev
