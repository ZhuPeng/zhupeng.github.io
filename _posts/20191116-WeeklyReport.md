**7、GitHub 密码防泄漏工具 **

虽然每个人都知道不要将密码、秘钥等信息放到仓库代码里面，但是密码泄露的事情其实一直在发生，简直就是防不胜防。

曾经年少无知的我就犯过这样的错误，以前调用 GitHub 的相关接口用到 Token，有时候会直接提交到代码仓库里面，不过机制的 GitHub 会直接给你发告警邮件，同时会废除你使用的这个 Token，避免了泄露。
